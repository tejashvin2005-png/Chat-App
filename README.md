# 💬 Chat Application

## 📌 Project Description

This is a real-time chat application that allows users to communicate instantly over the internet. The application provides a simple and interactive interface for sending and receiving messages between users. It is designed using modern web technologies to ensure fast, responsive, and reliable communication.

---

## 🚀 Features

* 🔹 Real-time messaging between users
* 🔹 User-friendly interface
* 🔹 Instant message updates (no page refresh)
* 🔹 Scalable backend using Node.js
* 🔹 Supports multiple users chatting simultaneously
* 🔹 Organized client-server architecture

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Real-time Communication: Socket.io

---

## ⚙️ How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/akanksha040905/chat-app.git
```

### 2️⃣ Navigate to project folder

```bash
cd chat-app
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the server

```bash
node server.js
```

### 5️⃣ Open in browser

Go to:

```
http://localhost:3000
```
